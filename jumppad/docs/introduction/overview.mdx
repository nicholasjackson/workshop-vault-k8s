# Vault Secrets Operator Workshop

Welcome to the Vault Secrets Operator (VSO) workshop! In this workshop, you will learn how to integrate HashiCorp Vault with Kubernetes using the Vault Secrets Operator.

## What You Will Learn

- How the Vault Secrets Operator works
- Configuring Vault authentication for Kubernetes workloads
- Syncing static secrets from Vault to Kubernetes
- Using dynamic secrets with the database secrets engine
- Best practices for secrets management in Kubernetes

## Prerequisites

This workshop assumes you have basic familiarity with:

- Kubernetes concepts (pods, secrets, service accounts)
- HashiCorp Vault basics (secrets engines, policies, authentication)
- Command line tools (kubectl, vault CLI)

## Workshop Environment

The workshop environment includes:

- A local Kubernetes cluster (K3s)
- HashiCorp Vault running in development mode
- The Vault Secrets Operator pre-installed
- A PostgreSQL database for dynamic secrets demos
